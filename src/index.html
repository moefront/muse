YMPlayer 4 Demo

Description

This is just a simple and diligent HTML5 Audio Player.

Support

YMPlayer supports browsers supported HTML 5 Audio API, such as Internet Explorer (10+), Mozilla Firefox, Chrome, Opera as well as Microsoft Edge.

Usage

If you are a user of YMPlayer, please read README.MD in the project root directory, after that you will know how to render a player to your page.

If you are a developer, please refer to Wiki on GitHub. It will tell you details of the project exactly.

Troubleshooting

Please go to GitHub and open an issue / pull a request whenever you face a problem or have some suggestions, or contact me at kirainmoe@gmail.com.
